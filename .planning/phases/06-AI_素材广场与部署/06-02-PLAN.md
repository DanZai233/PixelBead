   - Return Material array

   - URL: `${API_BASE_URL}/api/materials`

   - Fetch with POST method, headers: { 'Content-Type': 'application/json' }
   - Handle offline: if fetch fails (network error), throw new Error('Network error')
   - Parse JSON response with { id }
   - Return { id }

6. Error handling:
   - Try/catch around fetch calls
   - Re-throw errors with descriptive messages
   - Log errors to console.error

Export fetchMaterials, publishMaterial, Material.
