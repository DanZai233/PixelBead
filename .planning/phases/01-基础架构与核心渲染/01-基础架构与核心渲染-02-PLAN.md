  <name>Task 1: Create @pixelbead/shared-types package</name>
  <files>packages/shared-types/package.json, packages/shared-types/tsconfig.json, packages/shared-types/src/index.ts</files>
  <action>
    1. Create package directory and structure:
       - mkdir -p packages/shared-types/src

    2. Create packages/shared-types/package.json:
       - name: "@pixelbead/shared-types" (per D-02: package naming convention)
       - version: "1.0.0"
       - main: "./src/index.ts"
       - types: "./src/index.ts"
       - private: true (internal package)
       - dependencies: {} (no dependencies, pure types)

    3. Create packages/shared-types/tsconfig.json:
       - extends: "../../tsconfig.json" (or strict mode config if root not strict)
       - compilerOptions:
         - strict: true (per D-08: TypeScript strict mode)
         - noImplicitAny: true (per D-15: no any types)
         - outDir: "./dist"
         - moduleResolution: "node"

    4. Extract types from root types.ts to packages/shared-types/src/index.ts:
       - Export all type definitions:
         - export type ColorHex = string;
         - export type ColorSystem = 'MARD' | 'COCO' | '漫漫' | '盼盼' | '咪小窝';
         - export interface PaletteColor { hex: string; key: string; count?: number; }
         - export interface BeadGrid { name: string; description: string; grid: ColorHex[][]; size: number; }
         - export enum ViewType { TWO_D = 'TWO_D', THREE_D = 'THREE_D', SLICES = 'SLICES' }
         - export enum ToolType { PENCIL = 'PENCIL', ERASER = 'ERASER', FILL = 'FILL', PICKER = 'PICKER', LINE = 'LINE', RECT = 'RECT', CIRCLE = 'CIRCLE', SMART_PENCIL = 'SMART_PENCIL', HAND = 'HAND', SELECT = 'SELECT' }
         - export enum PixelStyle { CIRCLE = 'CIRCLE', SQUARE = 'SQUARE', ROUNDED = 'ROUNDED' }
         - export interface ToolInfo { type: ToolType; name: string; icon: string; shortcut: string; description: string; }
         - export enum AIProvider { OPENAI = 'OPENAI', OPENROUTER = 'OPENROUTER', DEEPSEEK = 'DEEPSEEK', VOLCENGINE = 'VOLCENGINE', GEMINI = 'GEMINI', CUSTOM = 'CUSTOM' }
         - export interface AIConfig { provider: AIProvider; apiKey: string; model?: string; baseUrl?: string; endpoint?: string; imageUrlModel?: string; }
         - export interface Selection { startRow: number; startCol: number; endRow: number; endCol: number; }
         - export interface PaletteConfig { selectedPreset: string; maxColors: number; mergeThreshold: number; showColorKeys: boolean; selectedColorSystem: ColorSystem; }

       - Export constants (as const where applicable):
         - export const DEFAULT_COLORS: ColorHex[] = [...]
         - export const AI_MODELS = { ... }
         - export const DEFAULT_ENDPOINTS = { ... }
         - export const TOOLS_INFO: ToolInfo[] = [...]
         - export const PIXEL_STYLES = [...]
         - export const PALETTE_PRESETS = [...]
         - export const BRUSH_SIZES = [...]

       - Use named exports only (per D-03: no default exports for consistency)

    Rationale: Shared types package is foundation for other packages. No dependencies means no circular dependency risk. Strict TypeScript catches type errors early (per D-15). Named exports enable tree-shaking (per D-03).
  </action>
  <verify>
    <automated>cd packages/shared-types && pnpm tsc --noEmit</automated>
  </verify>
