<task type="auto">
  <name>Task 2: Create bottom tab bar layout with Expo Router</name>
  <files>apps/mobile/app/(tabs)/_layout.tsx</files>
  <action>
Create apps/mobile/app/(tabs)/_layout.tsx for bottom tab bar navigation:

```typescript
import { Tabs } from 'expo-router';
import { MaterialIcons } from '@expo/vector-icons';

export default function TabLayout() {
  return (
    <Tabs
      screenOptions={{
        tabBarActiveTintColor: '#007AFF',
        tabBarInactiveTintColor: '#999',
        tabBarStyle: {
          height: 60,
          paddingBottom: 5,
          paddingTop: 5,
        },
        headerShown: false,
      }}
    >
      <Tabs.Screen
        name="editor"
        options={{
          title: 'Editor',
          tabBarIcon: ({ color, size }) => (
            <MaterialIcons name="edit" size={size} color={color} />
          ),
        }}
      />
      <Tabs.Screen
        name="gallery"
        options={{
          title: 'Gallery',
          tabBarIcon: ({ color, size }) => (
            <MaterialIcons name="photo-library" size={size} color={color} />
          ),
        }}
      />
      <Tabs.Screen
        name="settings"
        options={{
          title: 'Settings',
          tabBarIcon: ({ color, size }) => (
            <MaterialIcons name="settings" size={size} color={color} />
          ),
        }}
      />
    </Tabs>
  );
}
```

Design notes:
- `(tabs)` folder creates a tab navigator group (Expo Router pattern)
- Bottom tab bar with 3 tabs: Editor, Gallery, Settings
- MaterialIcons for tab icons
- 60px tab bar height (44px minimum + padding for iOS safe area)
- `headerShown: false` removes top header (cleaner creative tool look)
- Active color: #007AFF (iOS blue), inactive: #999 (gray)
