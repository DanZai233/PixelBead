    Create apps/mobile/src/hooks/useColorStatistics.ts:

    2. Import: { getColorCounts, mergeSimilarColors } from '@pixelbead/color-system'
    3. Export function:
       - useColorStatistics(): Returns { colorCounts, mergeThreshold, handleMergeSimilarColors, highlightColor }
    4. Calculate colorCounts:
       - Convert canvasStore.grid from Map to 2D array for getColorCounts
       - Call getColorCounts(grid) to get array of { hex, key, count }
       - Sort by count descending
       - Return via useMemo to prevent recalculation on every render
    5. Merge similar colors:
       - handleMergeSimilarColors(threshold: number):
         - Call mergeSimilarColors(colorCounts, threshold)
         - Map merged colors back to canvasStore.grid using findClosestColor
         - Update canvasStore.grid with merged colors
       - Note: threshold parameter is 0-50% (0.0 to 0.5)
    6. Highlight color:
       - highlightColor(hex: string):
         - Set canvasStore.highlightedColor to hex
         - CanvasGrid can use this to draw highlight overlay on matching pixels
         - Clear highlight by calling with null

    Design: Memoized color counts for performance, threshold slider for merging, highlight state for visual feedback.
