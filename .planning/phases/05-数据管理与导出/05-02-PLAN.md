  <name>Task 1: Create canvas export utilities</name>
  <files>
    apps/mobile/src/utils/canvasExport.ts
  </files>
  <action>
    Create apps/mobile/src/utils/canvasExport.ts:

    1. Import: Canvas, Rect, Circle, Skia, useFont, Image, Group, Text as SkiaText from '@shopify/react-native-skia'
    2. Import: * as FileSystem from 'expo-file-system'
    3. Import: * as Sharing from 'expo-sharing'

    4. Define TypeScript interfaces:
       - interface ExportOptions { showGrid: boolean; showColorCodes: boolean; mirror: 'none' | 'horizontal' | 'vertical'; exportSelection: boolean; }

    5. Create utility functions:
       - exportCanvasAsPng(canvasState: CanvasState, options: ExportOptions): Promise<string>
         - Determine export dimensions based on selection region or full grid
         - Create Surface using Surface.make(width, height)
         - Render pixels based on pixelStyle (CIRCLE/SQUARE/ROUNDED)
         - If showGrid: Draw grid lines with stroke
         - If showColorCodes: Draw hex codes on each pixel (use loaded font)
         - If mirror != 'none': Apply Group with transform (scaleX/Y = -1, translate)
         - Capture image using surface.toImage()
         - Convert to PNG using image.encode(Skia.ImageFormat.PNG)
         - Write to temporary file (FileSystem.documentDirectory + 'export.png')
         - Return file URI

       - shareExportedImage(fileUri: string): Promise<void>
         - Call Sharing.shareAsync(fileUri)
         - Handle permission errors

       - calculateExportRegion(canvasState: CanvasState, exportSelection: boolean): { x1: number; y1: number; x2: number; y2: number }
         - If exportSelection and selectionRegion exists: return selectionRegion
         - Else: return full grid bounds (0, 0, gridSize.width, gridSize.height)

       - renderGridLines(options: ExportOptions, width: number, height: number): React.ReactNode
         - Draw horizontal and vertical lines at each pixel boundary
         - Line color: light gray (#E0E0E0), stroke width: 1

       - renderColorCodes(canvasState: CanvasState, options: ExportOptions, font: any): React.ReactNode[]
         - Draw hex codes centered on each pixel
         - Font size: 8px, color: contrast-aware (white on dark pixels, black on light pixels)

    6. Load font: useFont(require('../assets/fonts/Roboto-Regular.ttf'), 12) for color codes

    7. Add JSDoc comments for all functions

    Note: Use Skia's off-screen rendering (Surface.make) to export image at full resolution, not screen capture.
  </action>
